# Project Todos

Each project has its own local todo list. Open it with **Todos** on the project row or from the project detail header.

## Creating and editing tasks

The popup contains the task list on the left and the editor on the right. A todo has:

- a title
- an optional description
- a status: **New**, **In progress** or **Done**
- a priority: **Low**, **Normal** or **High**

Select an existing task to edit it. Status can also be changed directly in the list.

## Sorting

Choose between manual order, status, priority, newest first or title. In manual mode, use the arrow buttons to move a task up or down. The manual order is stored with the project.

## Storage

Todos are part of the local CodeDeck configuration. They do not create or modify files inside the project folder. They are included when the CodeDeck configuration is exported.
